# qwen_oauth

Qwen OAuth support for Pi.

This extension adds a `qwen-oauth` provider for Qwen 3.6 Plus to Pi.

## Installation

### Install from GitHub

```bash
pi install git:https://github.com/redzrush101/qwen_oauth
```

### Clone locally and load from the directory

```bash
git clone https://github.com/redzrush101/qwen_oauth.git
cd qwen_oauth
pi -e .
```

## Usage

Start Pi with the extension installed or loaded, then authenticate:

```text
/login qwen-oauth
```

Then select the Qwen provider in Pi.
